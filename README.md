## 翻译解释
**新增**：新增的API  
**修改**：对API的更改  
**删除**：删除以前的API  
**起草**：新增的API，可能可以使用，但是***可能***会有Bug  
**弃用**：将API予以弃用，有时候可能是版本更迭，也有可能是有一定的Bug  
**去除起草标记**: 将以前起草的API标记进行去除，（予以肯定
